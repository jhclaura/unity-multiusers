# unity-multiusers
Multiuser between Unity and Three.js (web) using Socket.io with library BestHTTP

## Server
The server side scripts are in the **server** folder
````bash
npm install
nodemon server.js
````
## Web Version
Go to super rough webpage
````bash
localhost:3000
````

## Unity Version
1. Add **SteamVR**, **GVR**, and **BestHTTP** libraries
